# AGENT INSTRUCTIONS — Read this before doing anything in this repo

If you are an AI agent (SuperNinja, Copilot, Claude, Cursor, Gemini, etc.) and
you are about to read, edit, or build anything in this repository:

**Read `docs/AGENT_MEMORY.md` first.**

That file is the single source of truth for this project:
- what the project is and how it's laid out
- build / verify commands
- the Git + PR workflow you must follow
- the design system (use the CSS classes, not inline styles)
- the "de-slop" rules (how to not look AI-generated)
- UX and accessibility principles
- known no-gos
- **the Change Protocol** — the rule that *every* change, small or big,
  must update `docs/AGENT_MEMORY.md` and append to
  `change-made-after-submission.md`, then ship as a PR.

If `docs/AGENT_MEMORY.md` conflicts with any other doc in the repo,
`docs/AGENT_MEMORY.md` wins, and you should fix the other doc to match.

— RealLearn maintainers
